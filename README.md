# CamusAPI
Open API platform for Tsinghua University Developer.

## Usage

### pre install
`node v7.0` is required.

### install
```
npm install
```

### run
```
npm start
```

## Develop
```
npm run dev
```

### lint
```
npm run lint
```

you should pass the lint before every commit.
